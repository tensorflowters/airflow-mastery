## :rocket: Quick Start

Get up and running in under 5 minutes:

=== "macOS / Linux"

    ```bash
    curl -fsSL https://raw.githubusercontent.com/YOUR_ORG/airflow-mastery/main/scripts/quickstart.sh | bash
    ```

=== "Windows (PowerShell)"

    ```powershell
    irm https://raw.githubusercontent.com/YOUR_ORG/airflow-mastery/main/scripts/quickstart.ps1 | iex
    ```

This will:

1. :white_check_mark: Clone the repository
2. :white_check_mark: Install dependencies with **uv**
3. :white_check_mark: Start Airflow with Docker Compose
4. :white_check_mark: Open the Airflow UI and documentation
